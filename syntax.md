# Up Syntax

This file explains precisely Up's syntax

## Variables

Declare a variable :

```
# Default init
int a

# Init
int b = 42

# Auto type
$c = 42
```

## Types

Up types are not C types :

| Up Type | Description / Equivalent |
| ------- | ------------------------ |
| int | int32 |
| num | float32 |
| bool | uint8 |
| str | const char* (WIP) |
| nil | void |

## Literals

Here are some literals :

```
$imInt = 42
$imNum = 42.
$imStr = 'wow'
$imBool = yes
$imBool2 = no
```

## Operations

Operations are close to C operations :

```
a++

b -= -1

c /= 42
```


## Modules

To import modules :

```
use module
```

libc is used to have c headers :

```
use libc
```

## C Compatibility

To declare extern C functions :

```
cdef num cosf(num)
```
